# GrimoireTradeCalculator

## 概要

`GrimoireTradeCalculator` は、グリモアのトレードまたはリサイクルを用いて、指定した要求レベルのグリモアを生成できる組み合わせを計算する Windows フォームアプリケーションです。

## 特長

- レベル 1 から 10 までの所持グリモア数を入力
- 目標とする要求グリモアレベルを指定
- トレード（Trade）とリサイクル（Recycle）の2つの交換方式を選択
- 可能な組み合わせをすべて列挙

## 使い方

1. アプリケーション(GrimoireTradeCalculator.exe)を起動
2. 画面上部のコンボボックスで `トレード` または `リサイクル` を選択
3. レベル 1 から 10 までの所持グリモア数を入力
4. `要求グリモアレベル` に目標レベルを設定
5. `計算` ボタンを押す
6. 計算結果が `計算結果` 欄に表示される

## 入力項目

- `トレード` / `リサイクル` の交換方式
- `Lv1` から `Lv10` までの所持数
- `要求グリモアレベル`（1〜10）

## 出力

- 可能な組み合わせが各行ごとに配列形式で表示されます
- 組み合わせが存在しない場合は `トレードできません` と表示されます

## アルゴリズム

- `Calculator` クラスが組み合わせ計算を担当
- 各レベルのグリモア価値は、トレード時は前レベルの3倍、リサイクル時は前レベルの2倍を基準に計算
- 再帰的な探索で、所持数の範囲内に収まるすべての組み合わせを列挙

## 動作環境

- .NET 8 (.NET 8.0-windows7.0)
- Windows Forms 対応環境

## 開発者向け情報

- ソースコードは `GrimoireTradeCalculator/Calculator.cs` に計算ロジック
- `GrimoireTradeCalculator/Form1.cs` が UI イベント処理
- `GrimoireTradeCalculator/Form1.Designer.cs` がフォームの構成
